Notify-me

A simple tool to send notications and mails when a job is done

View project on GitHub

Introduction.

Do you run long processes on Linux Servers? Do you check these jobs regularly? What about get notified when the job succeed or failed.

Support

Currently, Notify-me supports SMTP Emails and push notifications on Andriod OS using Python Notifications.

Installation

simplely, run the following command and you will have Notify-me ready,

sudo pip install pushNotification notify-me

Configuration

You can configure the SMTP and push notification through /etc/notiy-me.cfg or ~/.notify-me.cfg
